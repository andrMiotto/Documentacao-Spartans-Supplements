Spartans Supplements API

Referência da API REST para integração com produtos, usuários e cupons.

Visão Geral

A API Spartans Supplements permite gerenciar produtos, usuários e cupons.

Base URL

https://spartans-supplements.onrender.com

Produtos

Endpoints para cadastro, listagem, atualização e exclusão de produtos.

POST /products/register

Registrar novo produto

GET /products/list

Listar todos os produtos

GET /products/list/{id}

Obter produto por ID

PUT /products/update/{id}

Atualizar produto

DELETE /products/delete/{id}

Remover produto

Exemplo de resposta

JSON 
{
  "id": 32,
  "nome": "Whey Protein Isolate",
  "preco": 205,
  "descricao": "Proteína de rápida absorção para ganho de massa muscular.",
  "peso": 1,
  "categoria": "Suplementos",
  "imagemUrl": "https://...",
  "quantidadeEstoque": 51,
  "calorias": 1100,
  "proteinas": 600,
  "carboidratos": 200
}

Usuários

CRUD de usuários da loja.

POST /user/create

Criar usuário

GET /user/list/{id}

Buscar usuário por ID

PUT /user/update/{id}

Atualizar usuário

DELETE /user/delete/{id}

Cupons

Gerenciamento de cupons de desconto.

POST /coupon/register

Registrar cupom

GET /coupon/list

Listar cupons

GET /coupon/list/{id}

Cupom por ID

PUT /coupon/update/{id}

Atualizar cupom

DELETE /coupon/delete/{id}

Excluir cupom

{
